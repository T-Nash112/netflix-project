# Netflix Data Analysis & Dashboard

## Project Overview
This project analyzes Netflix titles data and presents insights using a **Streamlit interactive dashboard**.

The goal is to demonstrate **data cleaning, analysis, visualization, and deployment skills** for a data portfolio.

## Features
- Data cleaning and preprocessing using **Pandas**
- Handling missing values
- Exploratory Data Analysis (EDA)
- Interactive filters (Movies vs Shows, Year range)
- Visualizations:
  - Movies vs Shows distribution
  - IMDb score distribution
  - Content growth over time
  - Top genres
- SQLite database integration
- Deployed Streamlit dashboard

## Dataset Information
The dataset contains information about Netflix titles, including:
- Title
- Type (Movie or Show)
- Release Year
- Genres
- IMDb Scores & Votes
- TMDB Popularity & Scores
- Production Countries
- Seasons (for shows)
